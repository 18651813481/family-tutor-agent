# 家庭辅导老师

面向中文家庭的可语音 AI 家教 Agent 知识库与协作模板。

这个项目适合已经会使用 Codex、Claude、ChatGPT 项目线程或类似 Agent 工具的用户。它帮助你搭建一套“学生辅导线程 + 家长管理线程 + 本地知识库 + TTS 语音朗读”的家庭学习协作系统。

核心目标不是让 AI 直接代做题，而是帮助孩子训练读题、思考、表达、检查和复盘能力。

## 适合谁

- 想给孩子建立长期 AI 家教陪练的家长。
- 熟悉 `AGENTS.md`、项目知识库、线程分工的 Agent 用户。
- 希望把教材、老师通知、错题和学情观察系统化沉淀的人。
- 需要中文为主、英语可朗读、能接 TTS 的家庭学习场景。

不适合直接把真实孩子资料公开上传。真实教材、试卷、老师通知和孩子学情应保存在你的私有工作区。

## 快速开始

1. 复制本仓库到你的本地项目。
2. 把 `examples/` 中的示例复制到 `knowledge-base/` 对应位置，并替换成你家孩子的信息。
3. 填写 `knowledge-base/student-profile.md`，记录年级、教材、学习目标和注意事项。
4. 建立两个线程：
   - 家长管理线程：用于规则设定、学习资料归档、学情复盘和复习计划。
   - 学生辅导线程：只对孩子说话，用引导式方式陪孩子学习。
5. 配置 TTS，让学生辅导线程能朗读老师回复；家长管理线程默认静音。
6. 每次正式辅导前，让 Agent 先读 `knowledge-base/data_structure.md`，再按问题类型读取最相关文件。
7. 每次出现新教材、老师通知、错题或学情观察时，按 `knowledge-base/ingest-protocol.md` 归档。

## 推荐目录

- `AGENTS.md`：项目级 Agent 行为规则。
- `docs/tts.md`：学生线程 TTS 语音朗读接口和行为要求。
- `knowledge-base/data_structure.md`：知识库检索入口。
- `knowledge-base/thread-policy.md`：家长线程和学生线程分工。
- `knowledge-base/tutoring-playbook.md`：引导式教学流程。
- `knowledge-base/ingest-protocol.md`：资料、错题和学情归档协议。
- `examples/`：可复制改写的虚构示例。

## 中文 Prompt 示例

### 家长管理线程开场

请把当前线程作为家长管理线程。你负责帮我维护家庭辅导老师项目：更新知识库、整理教材和老师通知、分析孩子学情、设计复习计划。不要用这个线程直接辅导孩子做题。涉及孩子个人资料时，默认只保存在本地私有知识库，不要建议公开上传。

### 学生辅导线程开场

这是学生辅导线程。请按本项目 `AGENTS.md` 和 `knowledge-base/thread-policy.md` 的学生辅导规则来陪我学习。不要直接代做作业；遇到题目先让我说想法，再给提示。不要使用代码块、灰色文本框或预格式化文本。学英语时先让我读英文，再解释中文。

### TTS 语音线程配置要求

请确认这个学生辅导线程可以被 TTS 朗读。TTS 只读 assistant 的自然语言回复，不读代码、命令、路径、日志、URL 或密钥。自然语言里的英文单词、短语和句子必须正常朗读，不能跳过英文只读中文。家长管理线程默认不要发声。

### 新资料归档请求

我上传了一张老师通知。请按 `knowledge-base/ingest-protocol.md` 处理：先判断资料类型，提取老师要求、学科、时间、任务范围和待确认信息，再告诉我应该写入哪些知识库文件。不要把待办完成状态直接等同于孩子已经掌握。

### 错题复盘请求

这是一道孩子刚做错的题。请先记录题目摘要、孩子原思路、卡住点、错因类型和掌握状态。只出现一次的错误先标为“观察中”或“待复查”，不要直接写成长期薄弱点。然后给一个下次可复查的小练习。

### 期末复习请求

请基于老师通知、教材进度、孩子错题和最近学情，做一个 7 天期末复习计划。每天只安排 2-3 个重点，优先复查孩子已经暴露的卡点。不要押题，不要把公开样卷当作真实考试范围。

## TTS 是核心能力

学生辅导线程建议接入 TTS。孩子听到老师逐句朗读，比只看文字更容易跟读和保持参与。

最低要求：

- 只让学生辅导线程发声。
- 家长管理线程默认静音。
- 英文单词、短语和句子不能被过滤掉。
- 英语材料先读英文，再读中文解释。
- 输出不要用代码块、灰色文本框或中英同行格式。

详细规则见 `docs/tts.md`。

## 隐私原则

- 不要把真实孩子姓名、学校、班级、试卷、老师通知、语音、照片和学习记录提交到公开仓库。
- 公开仓库只放模板、规则、空表和虚构示例。
- 私有知识库可以记录真实学情，但应只保存在你自己的设备或私有仓库。
- 原始图片、PDF、音频和数据库不要进入公开仓库。
