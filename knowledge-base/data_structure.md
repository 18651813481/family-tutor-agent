# knowledge-base 索引

先读本文件，再进入最相关的窄文件。不要一次性读取整个知识库。

## 必读入口

- `README.md`：知识库使用顺序和维护原则。
- `student-profile.md`：学生基础信息和学习目标。
- `thread-policy.md`：家长管理线程和学生辅导线程分工。
- `tutoring-playbook.md`：具体辅导流程和提示方式。
- `ingest-protocol.md`：新资料、错题和观察记录的归档规则。
- `source-register.md`：资料来源和可信度。

## 教材与进度

- `curriculum/README.md`：教材层说明。
- `curriculum/chinese.md`：语文教材、单元、作文和阅读。
- `curriculum/math.md`：数学教材、单元、题型和方法。
- `curriculum/english.md`：英语教材、词汇、听说和 TTS。
- `curriculum/science.md`：科学主题、实验和概念。

## 学情与错题

- `learner/student-learning-profile.md`：学生学习画像模板。
- `learner/subject-weakness-map.md`：学科薄弱点地图。
- `question-log.md`：错题、口头问题和卡点记录。

## 考试与复习

- `exams/exam-calendar.md`：考试、单元练习和复习节点。
- `exams/review-playbook.md`：考前复习方法。

## 检索规则

- 问作业：先查老师通知和教材进度。
- 问错题：先查 `question-log.md`。
- 问长期弱项：查 `learner/subject-weakness-map.md`。
- 问英语朗读：查 `docs/tts.md` 和 `curriculum/english.md`。
- 问考试复习：查 `exams/`。
- 需要原始图片、PDF 或音频时，只引用私有路径，不把原始资料提交到公开仓库。
