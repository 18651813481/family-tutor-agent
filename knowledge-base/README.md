# 家教知识库

这里保存长期辅导所需的整理信息。它不是原始资料仓库；真实照片、PDF、音频和数据库应放在私有资料目录，不提交到公开仓库。

## 使用顺序

1. 先读 `data_structure.md`，定位最相关的知识库文件。
2. 再读 `student-profile.md`，确认孩子背景和辅导边界。
3. 如果要归档新资料，读 `ingest-protocol.md`。
4. 如果涉及线程边界，读 `thread-policy.md`。
5. 如果涉及具体教学方式，读 `tutoring-playbook.md`。
6. 如果涉及教材、考试、老师通知、错题或学情，进入对应目录。

## 维护原则

- 记录事实和来源，不把推测写成事实。
- 老师通知、教材实物、作业和试卷优先于公开资料。
- 不把一次错误写成长期薄弱点。
- 不把任务完成状态直接等同于已经掌握。
- 每条错题记录都要有下次可复查的小动作。
